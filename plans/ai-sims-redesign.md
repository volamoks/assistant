# AI Sims Dashboard Redesign Plan
## Terraria-Style 2D Isometric Office

---

## Vision

Превратить примитивный 3D офис в красивую 2D изометрическую сцену в стиле Terraria/The Sims:
- Пиксель-арт графика с четкими линиями
- Детализированные комнаты с мебелью
- Анимированные агенты, ходящие между комнатами
- Красивое освещение и тени
- Частицы и эффекты (пар от кофе, свет от мониторов)

---

## Architecture Decision

**Технология**: CSS 3D Transforms + React (без Three.js)

**Почему не Canvas/SVG**:
- CSS проще для пиксель-арт стиля
- Легче делать hover-эффекты и клики
- Можно использовать CSS-анимации
- Быстрее рендерится

**Структура изометрии**:
```
Изометрическая проекция: rotateX(54.7356deg) rotateZ(45deg)
Или классическая: transform: rotateX(60deg) rotateY(0deg) rotateZ(-45deg)
```

## Component Structure

### 1. IsometricOffice (заменяет Office.jsx)
- Контейнер с CSS 3D transform
- Обрабатывает координаты изометрии
- Depth sorting для правильного наложения

### 2. IsometricRoom
- Отдельная tile-комната с:
  - Изометрическим полом (tile)
  - Стенами (с depth)
  - Мебелью (столы, стулья)
  - Декором (растения, картины)

### 3. PixelAgent (заменяет AgentSprite)
- 8-bit стиль персонаж
- Анимации: idle, walk, work, coffee
- Изометрическое позиционирование
- Меняет направление при движении

### 4. Furniture & Decor
- Столы, компьютеры, диваны
- Растения, лампы, кофемашина
- Все в изометрическом стиле

## Agent Animations

### States:
1. **idle** - стоит, дышит (микро-анимация)
2. **walk** - идет (4 кадра анимации)
3. **work** - сидит за столом, печатает
4. **coffee** - пьет кофе
5. **meeting** - сидит на совещании

### Colors:
- MAIN: Фиолетовый (#9B59B6)
- CODER: Синий (#3498DB)
- RESEARCHER: Зеленый (#2ECC71)
- ANALYST: Желтый (#F1C40F)

---

## Movement System

Агенты должны:
1. Получать маршрут от backend (room -> room)
2. Двигаться по изометрической сетке
3. Иметь плавную анимацию передвижения
4. Менять направление спрайта

Pathfinding: простой A* или прямой путь между центрами комнат

---

## Visual Effects

1. **Shadows**: мягкие тени под агентами и мебелью
2. **Lighting**: свет от ламп, мониторов
3. **Particles**:
   - Пар от кофе
   - Пыль в лучах света
   - Уведомления над головой агента
4. **Hover effects**: подсветка при наведении
