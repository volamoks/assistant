# PM Toolkit

General-purpose product management skills applicable across industries including FinTech.

## Included Skills

### 1. stakeholder-map — Stakeholder Mapping
Identify and map key stakeholders for your product.

**Usage:**
```
Use skill: stakeholder-map
Args: New payment integration project
```

**Output:**
- Stakeholder matrix (power vs. interest)
- Communication plan
- Engagement strategies

---

### 2. retro — Sprint Retrospective
Facilitate effective retrospectives.

**Usage:**
```
Use skill: retro
Args: Sprint 23 retrospective for checkout team
```

---

### 3. summarize-meeting — Meeting Summaries
Summarize meeting notes and action items.

**Usage:**
```
Use skill: summarize-meeting
Args: [paste meeting transcript or notes]
```

---

### 4. grammar-check — Grammar & Style Check
Review and improve writing.

**Usage:**
```
Use skill: grammar-check
Args: [paste text to review]
```

---

### 5. review-resume — Resume Review
Review and improve resumes.

**Usage:**
```
Use skill: review-resume
Args: [paste resume or upload file]
```

---

## FinTech-Specific Considerations

When using these general skills in FinTech context, consider:

**Stakeholder Mapping:**
- Include compliance and legal teams
- Map banking partners
- Consider regulatory relationships

**Retrospectives:**
- Include security incidents in review
- Discuss compliance learnings
- Review fraud prevention effectiveness

**Meeting Summaries:**
- Document regulatory decisions
- Track compliance action items
- Record risk assessments
