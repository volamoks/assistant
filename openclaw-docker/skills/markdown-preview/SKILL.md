# Markdown Preview

## Что делает
- Запускает локальный сервер
- Рендерит markdown в реальном времени
- Авто-обновление при изменении файла
- Поддержка Mermaid диаграмм

## Команды

### Предпросмотр
```
открой preview для этой заметки
запусти markdown viewer
```

### Редактирование
```
редактирую с предпросмотром
покажи рендер в реальном времени
```

## Интеграция
Открывает браузер на localhost:3000 (или другой порт)
