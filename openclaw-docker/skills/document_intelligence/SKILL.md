# Document Intelligence

Use this tool to read and analyze complex documents (PDFs with tables, Word docs, Excel spreadsheets).
It returns a structured Markdown representation of the document.

**Usage:**
Provide the absolute path to the file within the container (e.g., `/data/obsidian/path/to/doc.pdf`).
