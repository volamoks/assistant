# Documentation Writer

## Что делает
- Пишет README для проектов
- Генерирует API документацию
- Создаёт troubleshooting гайды
- Пишет onboarding документы для команды

## Команды

### README
```
напиши README для проекта X
обнови документацию проекта
```

### API Docs
```
документируй этот API
создай спецификацию endpoints
```

### Гайды
```
напиши гайд по онбордингу
создай troubleshooting для ошибки X
```

## Формат
- Markdown
- Value-first заголовки
- Runnable примеры кода
- Active voice, present tense
