# Qwen-Agent Skill

## Описание
Skill для интеграции Qwen-Agent фреймворка от Alibaba в OpenClaw. Предоставляет возможности функционального вызова (function calling), интерпретатора кода (code interpreter), RAG и других возможностей Qwen-Agent.

## Триггеры
- "qwen" - основной вызов Qwen-Agent
- "code interpreter" - использование интерпретатора кода
- "browserqwen" - использование браузерного ассистента

## Установка зависимостей
Для работы этого skill необходима установка пакета qwen-agent. В зависимости от конфигурации системы могут потребоваться дополнительные зависимости:

```bash
pip install -U "qwen-agent[gui,rag,code_interpreter,mcp]"
```

## Конфигурация
Для использования облачного сервиса DashScope от Alibaba Cloud необходимо указать API ключ:

```bash
export DASHSCOPE_API_KEY=ваш_ключ_api
```

## Возможности
- Function Calling - вызов внешних функций и инструментов
- Code Interpreter - выполнение кода в песочнице
- RAG - поиск и анализ документов
- Browser Assistant - взаимодействие с веб-сайтами
- MCP - взаимодействие с внешними серверами через Model Context Protocol

## Безопасность
При использовании Code Interpreter рекомендуется запускать его в изолированной среде (Docker-контейнере) для обеспечения безопасности выполнения пользовательского кода.
