# API Docs Generator

## Что делает
- Сканирует код на наличие роутов
- Генерирует OpenAPI 3.0 спецификацию
- Создаёт Markdown документацию
- Добавляет примеры запросов/ответов

## Команды

### Генерация
```
сгенерируй API docs для проекта
создай OpenAPI спецификацию
```

### Обновление
```
обнови документацию API
пересгенерируй docs после изменений
```

## Формат вывода
- OpenAPI 3.0 (YAML/JSON)
- Markdown с примерами
- Postman коллекция (опционально)
