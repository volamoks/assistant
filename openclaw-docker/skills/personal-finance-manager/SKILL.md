# Personal Finance Manager

## Что делает
- Управление бюджетом по категориям
- Лимиты и алёрты
- Месячные summaries
- Локальное хранение (приватно)

## Команды

### Бюджет
```
установи бюджет на еду 500000
сколько осталось в категории "Развлечения"
```

### Алерты
```
предупреди если потрачу больше 80% бюджета
```

### Отчёты
```
месячный отчёт за март
сравнение с прошлым месяцем
```

## Хранение
SQLite: ~/.openclaw/pfm.db
