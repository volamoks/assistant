# Agent Money Tracker

## Что делает
- Запись расходов через NLP («потратил 50к на еду»)
- Автоматическая категоризация
- Трекинг бюджета по категориям
- Отчёты и тренды

## Команды

### Записать расход
```
потратил 50000 на продукты
купил бензин за 3000
обед 25000 сум
```

### Записать доход
```
получил зарплату 10000000
премия 2000000
```

### Отчёты
```
сколько потратил в этом месяце
покажи расходы по категориям
сколько осталось до конца месяца
```

## Хранение
SQLite база: ~/.openclaw/money-tracker.db

## Настройка
Работает из коробки. База создаётся автоматически.
